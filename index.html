BehindTheNet.org

This domain once hosted the blog of Josh C., software developer, Virginia Tech alumnus and Carolina Hurricanes/Washington Nationals fan in Charlottesville, VA.

I'm still around, but the site, not so much. During blogging's peak, I had lots of time to write and maintain my own web space. Life moved on, as did I. Maintaining the archive as self-hosted blogging software required regular security updates, then third-party hosting services failed or shut down, became more trouble than it was worth. And I'm not the same person as I was in 2003 with all the same opinions that I had then.

I might choose to write again sometime, and it's possible I'd do so here. But the best places to find me for now are:

  • Twitter @joshcvt is for off-the-cuff talk and pressure venting, more sports than not. If locked down, I'll probably approve a follow request if I recognize you.
  • Goodreads joshcvt gets reads and reviews since fall 2014, and some back content as well.
  • GitHub joshcvt is where I work on some dev projects I don't mind open-sourcing. The current highlight is natinal, a Python monitoring system for MLB.com live data feeds. And of course, this site is itself hosted as GitHub Pages site.

The substantive way to reach me is by email.

Thanks for visiting.
